address.new

URL

https://www.bitmex.com/api/v1/address

HTTP Method

POST

Response Type

Address

Parameters

  • Form: currency String

    Currency of the address. Options: `XBt`, `USDt`

  • Form: network String

    Selected network.

  • Form: address String

    Destination Address.

  • Form: name String

    Name of the entry, eg. 'Hardware wallet'.

  • Form: note String

    Optional annotation.

  • Form: skipConfirm Boolean

    Skip e-mail confirmations for transfers to this address. Will require an email confirmation after creation.

  • Form: skip2FA Boolean

    Skip 2FA confirmations for transfers to this address. Will require an email confirmation after creation.

  • Form: memo String

    Destination Memo.
