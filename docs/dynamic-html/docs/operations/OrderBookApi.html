This is the API

orderBook.get

URL

https://localhost/api/v1/orderBook

HTTP Method

GET

Response Type

List

Parameters

  • Query: symbol String

    Instrument symbol. Send a series (e.g. XBT) to get data for the nearest contract in that series.

  • Query: depth BigDecimal

    Orderbook depth.

orderBook.getL2

URL

https://localhost/api/v1/orderBook/L2

HTTP Method

GET

Response Type

List

Parameters

  • Query: symbol String

    Instrument symbol. Send a series (e.g. XBT) to get data for the nearest contract in that series.

  • Query: depth BigDecimal

    Orderbook depth per side. Send 0 for full depth.
