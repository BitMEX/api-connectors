This is the API

position.get

See <a href=\"http://www.onixs.biz/fix-dictionary/5.0.SP2/msgType_AP_6580.html\">the FIX Spec</a> for explanations of these fields.

URL

https://localhost/api/v1/position

HTTP Method

GET

Response Type

List

Parameters

  • Query: filter String

    Table filter. For example, send {\"symbol\": \"XBTUSD\"}.

  • Query: columns String

    Which columns to fetch. For example, send [\"columnName\"].

  • Query: count BigDecimal

    Number of rows to fetch.

position.isolateMargin

URL

https://localhost/api/v1/position/isolate

HTTP Method

POST

Response Type

Position

Parameters

  • Form: symbol String

    Position symbol to isolate.

  • Form: enabled Boolean

    True for isolated margin, false for cross margin.

position.transferIsolatedMargin

URL

https://localhost/api/v1/position/transferMargin

HTTP Method

POST

Response Type

Position

Parameters

  • Form: symbol String

    Symbol of position to isolate.

  • Form: amount BigDecimal

    Amount to transfer, in Satoshis. May be negative.

position.updateLeverage

URL

https://localhost/api/v1/position/leverage

HTTP Method

POST

Response Type

Position

Parameters

  • Form: symbol String

    Symbol of position to adjust.

  • Form: leverage Double

    Leverage value. Send a number between 0.01 and 100 to enable isolated margin with a fixed leverage. Send 0 to enable cross margin.

position.updateRiskLimit

URL

https://localhost/api/v1/position/riskLimit

HTTP Method

POST

Response Type

Position

Parameters

  • Form: symbol String

    Symbol of position to update risk limit on.

  • Form: riskLimit BigDecimal

    New Risk Limit, in Satoshis.
