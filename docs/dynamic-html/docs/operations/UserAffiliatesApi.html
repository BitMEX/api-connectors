This is the API

userAffiliates.get

URL

https://www.bitmex.com/api/v1/userAffiliates

HTTP Method

GET

Response Type

List

Parameters

  • Query: depth Double

    the depth of affiliates to return. Eg depth = 2 would return direct affiliates and their affiliates

  • Query: targetAccountId Double

    AccountId of Sub-Affiliate Account

  • Query: selectUserId Double

    User id of result array to keep
